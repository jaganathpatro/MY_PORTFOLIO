ESP32 LED Pattern Display

Project Overview

This ESP32-based project demonstrates multiple LED patterns using digital output pins. The patterns include running lights, blinking effects, and symmetrical animations, which can be extended using WiFi control for real-time changes.

Technical Details

  • Microcontroller: ESP32
  • IDE: Arduino IDE
  • LEDs: 7x LEDs with 220Ω resistors
  • Power: USB 5V / Battery
  • Control: Hardcoded + WiFi (optional)

Challenges & Solutions

Precise timing between LED transitions was optimized using `delay()` and non-blocking code structures. Debouncing switches was also considered for real-time control.

Future Improvements

Integration of a mobile app interface for pattern selection and custom animation upload via OTA update.

Set-Reset and Oscillating Circuit

Project Overview

This hardware project combines a Set-Reset latch (using logic gates or transistors) with a simple oscillating circuit powered by DC. It demonstrates the principles of digital memory and signal generation.

Technical Details

  • Components: NPN Transistors, Resistors, Capacitors
  • Logic: SR latch using NOR/NAND gates
  • Oscillation: Astable multivibrator using BJTs
  • Power: 9V Battery or DC supply
  • Tools: Breadboard and Multimeter

Challenges & Solutions

Stability of oscillation was a key challenge. Correct resistor-capacitor combinations ensured smooth clock pulses. Using LEDs helped visualize signal output from the circuit.

Future Improvements

Plan to add push-button debounced triggers for SR control and a logic analyzer for real-time visualization of output states.
